参数说明

名称 说明 默认值 是否选填 接受类型 版本号
containerSelector 用来包裹mPlayer播放器的容器 无 否 jQuery Selector 1.3.3
playRotate 播放时专辑图片是否旋转 true 是 Boolean 1.3.3
useDefaultStyle 您是否正在使用默认样式（如果您对样式有了大幅度调整或重写了样式，请将此项设置为false，否则您的排版可能会错乱，详情查看修改注意事项） true 是 Boolean 1.3.3
lrcHeight 歌词区域的高度（当useDefaultStyle设置为false时本配置项无效） 160 是 Int 1.3.3
beforePlay 每首歌播放开始前的回调函数，接受一个参数作为歌曲信息 无 是 Function 1.3.3
canPlay 歌曲缓冲到可以播放时的回调函数（用户跳跃播放时间时同样会触发），接受一个参数作为歌曲信息 无 是 Function 1.3.3
autoPlay 初始化完成后是否自动播放 true 是 Boolean 1.2.0
songList 歌曲播放列表 无 否 Array 1.0.0
playMode 播放模式（0代表列表循环，1代表单曲循环，2代表随机播放） 0 是 Int 1.0.0
playList 第一首播放的歌曲所属的列表序号 0 是 Int 1.0.0
playSong 第一首播放的歌曲在列表中的索引（忽略basic） 0 是 Int 1.0.0

说明

如果您对mPlayer的样式不满意，您可以对其进行修改，下面会列出所有的CSS选择器及修改注意事项

选择器

序号 说明 选择器 备注
1 专辑图片的容器 .mp-cover 其中img标签请用.mp-cover img选择
2 处于当前状态的歌词 .mp-lrc-current -
3 非当前状态的歌词 .mp-lrc -
4 歌曲的已播放时间和总时间 .mp-progress-time -
5 播放模式容器 .mp-modebox 其中的各个按钮为a标签 c* f* h*
6 列表显示开关 .mp-menu-switch c* f* h*
7 静音按钮 .mp-vol-mute f* h*
8 音量调节 .mp-vol-range -
9 播放按钮 .mp-button-play f* h*
10 下一首按钮 #mp-button-next 不推荐使用id写样式，尽量使用.mp-buttonbox a给所有控制按钮设置样式，然后覆盖播放按钮的样式 f* h*
11 上一首按钮 #mp-button-prev 同上
12 总进度条 .mp-progress-total -
13 已播放的进度条 .mp-progress-current -
14 歌名、歌手 .mp-playing-name 歌名为h3标签，歌手为h3中嵌套的small标签
15 当前显示的列表名称 .mp-list-title -
16 列表项 .mp-list-select-ul li h*
17 当前正在播放的歌曲 .mp-list-song-current -
18 正常歌曲 .mp-list-song h*
19 所有歌曲列表 .mp-list-select-on .mp-list-select-ul o*
20 整个列表容器 .mp-list-on .mp-listbox o*

c*：该元素被点击后将添加.mp-deep类

f*：请用font-size和color改变图标的大小和颜色

h*：该元素有鼠标划过效果

o*：该元素有显示和隐藏效果，点击开关后会给该元素的父元素增加一个class，修改时应注意显示和隐藏问题

有部分元素没有在此列出，如要使用请使用审查元素查看类名或ID

修改注意事项
