# mPlayer音乐播放器

***
请在您的网页中插入以下内容
```html 
<script>new Mplayer(mplayer_song);</script>
```
***

> ## 优点
> 
> 1. 歌词显示功能
> 2. 三种播放模式切换
> 3. 多列表支持
> 4. 易于使用，即使不会任何JS都能使用
